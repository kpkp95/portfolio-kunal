## Domain RAG Assistant
- Multi-domain RAG chatbot system deployed on AWS EC2
- Technologies: Python, LangChain, Pinecone, Redis, Docker, GitHub Actions CI/CD, FastAPI, AWS EC2
- Architecture:
  - Indexes 384-dimensional sentence embeddings in Pinecone
  - Metadata-based domain filtering (Medical, Machine Learning, LLM domains)
  - Redis-backed conversation memory for session persistence
  - Source citations returned per response (file, page, domain, chunk ID)
- Performance: Achieved 12/12 retrieval evaluation tests (perfect accuracy)
- Deployment: AWS EC2 via Docker + GitHub Actions CI/CD with self-hosted GitHub Actions runner
- Live demo available at: http://54.172.244.2:8080/
